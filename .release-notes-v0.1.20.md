## Highlights

- Improve personal WeChat QR binding, including safer QR preview handling in Electron.
- Add native personal WeChat QR login and webhook parsing support.
- Add email channel support with IMAP/SMTP workflows.
- Improve channel logs, message display, slash/control commands, and tool error handling.
- Improve startup performance, pipeline UX, logging, and memory robustness.

## Packaging

GitHub Actions will build and attach Windows, macOS, and Linux artifacts to this release.
