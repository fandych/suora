快速开始

三步上手

  1. 从 GitHub Releases 下载适合 Windows、macOS 或 Linux 的安装包。
  2. 打开 Models 页面，配置至少一个模型提供商或本地接口。
  3. 根据你的工作方式，从 Chat、Documents、Pipelines 或 Timer 开始使用。
