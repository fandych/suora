Setup / 上手

Install in 3 steps

A short operational path for turning the release build into a working desktop setup.

  1. Pick the matching package. Windows users can start with `.exe`, macOS with `.dmg` or `.zip`, and Linux with `.AppImage`, `.deb`, or `.rpm`.
  2. Configure a model provider. Open the Models view, add an API key or local endpoint, then enable at least one model for chat and agents.
  3. Choose your working mode. Start with Chat for direct interaction, Documents for knowledge work, or Pipelines and Timers for automation flows.
