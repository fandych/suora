Quick start

Three-step setup

  1. Download the matching package for Windows, macOS, or Linux from the latest GitHub release.
  2. Open the Models view and configure at least one provider or local endpoint.
  3. Choose Chat, Documents, Pipelines, or Timers based on the work style you want first.
