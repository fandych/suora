# 从这里开始了解 Suora

Suora 是一个基于 Electron 的 AI 工作台，覆盖聊天、文档、Agent、技能、流水线、定时任务、渠道接入、MCP 集成和工作区设置。现在 GitHub Pages 站点已经改成 **Docusaurus + React + MDX**，让公开文档入口更像正式文档站。

## 最短路径

1. 先从 [GitHub Releases](https://github.com/fandych/suora/releases/latest) 下载最新版本。
2. 启动 Suora 后，在 **Models** 中配置至少一个模型提供商。
3. 根据你的使用场景，从 **Chat**、**Documents**、**Pipeline** 或 **Timer** 先开始。

## 这个 Pages 站点负责什么

- 用更正式的文档站方式替代旧的静态 HTML 首页。
- 保持 **单页面单语言**，通过页头按钮切换语言。
- 让 GitHub 仓库地址一直保持醒目。
- 在部署 GitHub Pages 时自动打包生成站点，而不是手工提交产物。
