## 最短路径

1. 先从 [GitHub Releases](https://github.com/fandych/suora/releases/latest) 下载最新版本。
2. 启动 Suora 后，在 **Models** 中配置至少一个模型提供商或本地模型端点。
3. 想先对话就进入 **Chat**，想先整理本地资料就进入 **Documents**。
4. 当你需要重复执行任务时，再使用 **Pipeline** 或 **Timer**。

## 你现在就能做什么

- 用 **Chat** 处理多会话对话、附件和工具协作。
- 用 **Documents** 整理本地笔记、文件夹、双向链接和图谱上下文。
- 用 **Agents** 和 **Skills** 组合出可复用的工作方式。
- 用 **Pipeline** 和 **Timer** 自动化重复步骤和定时任务。
- 需要外部集成时，再接入 **Channels** 和 **MCP**。
