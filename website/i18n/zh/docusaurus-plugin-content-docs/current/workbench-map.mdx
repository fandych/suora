# 工作台地图

Suora 是一个多模块桌面工作台，下面这些路由都已经属于当前产品范围。

<div className="doc-chip-row">
  <div className="doc-chip"><strong>交互层</strong><span>Chat · Models · Agents · Settings</span></div>
  <div className="doc-chip"><strong>知识层</strong><span>Documents · Skills</span></div>
  <div className="doc-chip"><strong>自动化</strong><span>Pipeline · Timer</span></div>
  <div className="doc-chip"><strong>外部连接</strong><span>Channels · MCP</span></div>
</div>

## 顶层路由

- `/chat`
- `/documents`
- `/pipeline`
- `/models/:view`
- `/agents`
- `/skills`
- `/timer`
- `/channels`
- `/mcp`
- `/settings/:section`

## 模块分组

| 分组 | 路由 | 主要用途 |
| --- | --- | --- |
| 与 AI 协作 | `/chat`、`/models/:view`、`/agents`、`/settings/:section` | 对话、模型配置、Agent 管理，以及工作区偏好设置 |
| 知识与自动化 | `/documents`、`/skills`、`/pipeline`、`/timer` | 本地文档、技能复用、流程编排和定时执行 |
| 连接外部系统 | `/channels`、`/mcp` | 消息渠道接入与 MCP 服务连接 |

## 默认跳转

- `/` 会跳转到 `/chat`
- `/models` 会跳转到 `/models/providers`
- `/settings` 会跳转到 `/settings/general`

## 设置分区

- `general`
- `security`
- `voice`
- `shortcuts`
- `data`
- `logs`
- `system`
