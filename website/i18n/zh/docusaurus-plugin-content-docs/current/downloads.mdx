# 下载说明

请把 [GitHub Releases](https://github.com/fandych/suora/releases/latest) 作为安装包和版本说明的唯一准入口。

## 部署方式

旧的 Pages 首页是直接提交到 `/docs` 的静态 HTML。现在改为在 GitHub Pages 工作流里构建 Docusaurus 站点，再把构建产物部署出去。
