## 打包版本

- **Windows**：安装版和便携版 `.exe`
- **macOS**：`.dmg` 和 `.zip`
- **Linux**：AppImage、DEB 和 RPM 包

## 从源码运行

```bash
npm install
npm run dev
```

如果你想参与开发、阅读代码，或验证未发布改动，可以直接从源码运行。
