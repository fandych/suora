## 语言策略

每一个渲染出来的 Pages 页面只保留一种语言，需要切换时通过页头按钮进入对应语言版本。

## 维护策略

现在仓库只保留一套更收敛的主文档：

- 中文 / 英文各一份主用户指南
- 中文 / 英文各一份主技术文档
- 少量仍在被引用、且明确有维护价值的专题文档

一次性升级报告、历史专项审计文档，以及无人维护的平行语言副本都已经移除，目的是让剩余文档更容易长期保持可信。
