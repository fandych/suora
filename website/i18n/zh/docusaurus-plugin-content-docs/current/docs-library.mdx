## 快速选择

- 想先看整体能力范围：从 **README** 或 **FEATURES.md** 开始。
- 想看安装、上手和日常使用：打开对应语言的 **用户指南**。
- 想看架构和运行机制：打开 **技术文档**。
- 想看文档智能规划素材：打开 **LLM Wiki 能力参考**。
- 想看测试、CI 或集成说明：使用 **测试说明** 和 **渠道集成说明**。

## 文档覆盖范围

当前仓库文档覆盖产品使用、实现细节、测试验证、渠道接入和需求范围。安装包下载请使用单独的下载页或 GitHub Releases。
