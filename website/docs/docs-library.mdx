## Language policy

Each rendered Pages page stays in a single language. Use the header buttons to switch locales instead of stacking English and Chinese on the same page.

## Maintenance policy

The repository now keeps a smaller curated documentation set:

- one primary user guide in Chinese and one in English
- one primary technical guide in Chinese and one in English
- a few focused topic guides that are still linked and actively maintained

Historical one-off reports and unmaintained language copies are intentionally removed so the remaining manuals stay easier to trust.
