## Quick picks

- Need the broad product overview? Start with **README** or **FEATURES.md**.
- Need end-user setup and workflows? Open the **User guide** in your preferred language.
- Need architecture or runtime details? Open the **Technical docs**.
- Need document-intelligence planning notes? Open the **LLM Wiki capability reference**.
- Need test, CI, or integration details? Use the **Testing guide** and **Channel integration** docs.

## Coverage

The repository documentation currently covers product usage, implementation details, testing, channel setup, and scope requirements. For packaged app downloads, use the separate Downloads page or GitHub Releases.
