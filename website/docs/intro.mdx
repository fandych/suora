## Fast path

1. Download the latest package from [GitHub Releases](https://github.com/fandych/suora/releases/latest).
2. Launch Suora and configure at least one provider or local endpoint in **Models**.
3. Start in **Chat** for conversations or **Documents** for local notes and context.
4. Add **Pipeline** or **Timer** once you want repeatable automation.

## What you can do right away

- Use **Chat** for multi-session conversations, attachments, and tool-assisted work.
- Use **Documents** to keep local notes, folders, backlinks, and graph-aware context.
- Use **Agents** and **Skills** to create reusable behaviors for different tasks.
- Use **Pipeline** and **Timer** to automate repeatable steps and scheduled runs.
- Use **Channels** and **MCP** when you need outside integrations.
