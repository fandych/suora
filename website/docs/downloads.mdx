## Packaged releases

- **Windows**: installer and portable `.exe`
- **macOS**: `.dmg` and `.zip`
- **Linux**: AppImage, DEB, and RPM packages

## Run from source

```bash
npm install
npm run dev
```

Use source mode when you want to develop, inspect the codebase, or test unreleased changes.
