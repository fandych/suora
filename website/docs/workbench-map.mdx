## Route surface

The current top-level routes include:

- `/chat`
- `/documents`
- `/pipeline`
- `/models/:view`
- `/agents`
- `/skills/:view`
- `/timer`
- `/channels`
- `/mcp`
- `/settings/:section`

## Why this matters for documentation

The homepage should feel like a hub for a real workbench, not a single-product marketing splash. That is why the new Pages design emphasizes:

- module navigation
- curated documentation entry points
- release downloads
- a persistent repository entry
