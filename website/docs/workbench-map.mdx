# Workbench map

Suora is a multi-module desktop workbench. These routes are already part of the current product surface.

## Top-level routes

- `/chat`
- `/documents`
- `/pipeline`
- `/models/:view`
- `/agents`
- `/skills`
- `/timer`
- `/channels`
- `/mcp`
- `/settings/:section`

## Module groups

| Group | Routes | What it covers |
| --- | --- | --- |
| Work with AI | `/chat`, `/models/:view`, `/agents`, `/settings/:section` | Conversations, model setup, agent configuration, and workspace preferences |
| Build knowledge and automation | `/documents`, `/skills`, `/pipeline`, `/timer` | Local notes, reusable skills, workflow automation, and scheduled actions |
| Connect external systems | `/channels`, `/mcp` | Messaging integrations and MCP server connections |

## Default route behavior

- `/` redirects to `/chat`
- `/models` redirects to `/models/providers`
- `/settings` redirects to `/settings/general`

## Settings sections

- `general`
- `security`
- `voice`
- `shortcuts`
- `data`
- `logs`
- `system`
